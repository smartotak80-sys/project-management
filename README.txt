Barakuda Stake RP — Static site (demo)
=====================================

Файли в архіві:
- index.html — головна сторінка
- styles.css — стилі
- script.js — клієнтська логіка (localStorage, UI)
- assets/ — тека з зображеннями (герой, галерея, placeholder)

Як запустити у Visual Studio / Visual Studio Code:
1. Розпакуйте архів у папку проекту.
2. Відкрийте папку у Visual Studio Code (File → Open Folder).
3. Рекомендую встановити розширення Live Server (Ritwick Dey).
4. Відкрийте index.html і натисніть "Go Live" (кнопка у правому нижньому куті).
5. Сайт відкриється у браузері. Всі дані (учасники, новини, галерея, заявки) зберігаються локально у localStorage для демонстрації.

Примітки:
- Це статичний фронтенд — для зберігання заявок/даних на сервері потрібен бекенд (наприклад Firebase, Google Sheets API або простий Node.js сервер).
- Ви можете редагувати стилі у styles.css та замінити зображення у папці assets.
